This is the Usersnap JQuery Example Page

For this example you will need an Usersnap account, create a FREE trial on http://usersnap.com/signup.

Please create a usersnap APIKey here and enter it to the textbox below. Please be sure that you have configured the right domain in your api configuration page.

Init usersnap

call $.usersnap({apiKey: '[your-api-key]', emailBox: true, usersnapLoaded: function() {alert('Usersnap is ready!')}});

show usersnap

call $.usersnap('show');
